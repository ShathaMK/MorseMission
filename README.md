## 🎮 Levels

**🔤 Level 1 — Learn the Alphabet**
Browse the full Morse Code alphabet and tap letters to hear their sound.

**🔐 Level 2 — Encode & Decode Secret Messages**
Type a message to encode it, or receive a transmission and decode it back.

**🕵️ Level 3 — Get on the Mission. Find the Spy.**
A spy is hiding at WWDC25 — use everything you've learned to catch them.


---



## 🚀 How to Run

1. Open `MorseMission.swiftpm` in **Swift Playgrounds** or **Xcode 16+**
2. Select an iPad simulator
3. Hit Run ▶️
